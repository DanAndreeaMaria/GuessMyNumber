I present to you a fun project, a game inspired from the 80's retro style.

As the name says, the goal for this project is to simply guess a 'secret number' 
which is between 1 and 20. So, in the input field we can type a number, we click 
'Check' and then on the right side we get a message which indicates hints about the 
secret number value. Lower on the right, we have the 'Score' and 'High Score' values which makes the interface look more like a game!
The 'Score' decreases by 1 each time we get the wrong number and if we really fail guessing, we'll get the message 'You lost the game!'. The game also saves the highest score registered by the player, I think this makes it really competitive!
In the end, we have the 'Again' button on top left corner which initiates a new game, reminding the early highest score. This is really cool, right? JavaScript is a really cool programming language and I'll focus more on the code rather on the HTML and CSS this time, but we'll capture everything, I promise!
So, here we will learn how to manipulate the DOM and how to change values which are initiated in HTML and how to change CSS background color via JavaScript.
